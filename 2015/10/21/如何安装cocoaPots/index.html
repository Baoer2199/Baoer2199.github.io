如何安装cocoaPots

发表于 2015-10-21

##关于如何安装CocoaPods

1.可以参考这个网页的教程：http://code4app.com/article/cocoapods-install-usage

2.按照以下步骤进行安装：

$：gem sources --remove https://rubygems.org
等有反应之后敲入以下命令
$gem sources -a http://ruby.taobao.org/

为了验证你的Ruby镜像是并且仅是taobao,可以使用以下命令查看

$gem source -l

只有在终端出现下面的文字才表明你上面的命令式成功的

*** CURRENT SOURCES***
http://ruby.taobao.org/

上面所有的命令完成之时，在终端上运行

$ sudo gem install cocoapods

这时你就可以使用这个命令来添加想要的第三方库

$ pod search AFNetworking

然后在项目中 用vim编辑器打开pot

$ vi pot

将得到的版本号粘贴复制进来 保存退出以后进行安装 也就是键入以下命令

$ pod install
